# AI Content Co-Pilot Project Context

This project is an AI-powered content generation tool built with Kiro assistance.

## Project Overview
- **Purpose**: Generate optimized social media content using Google Gemini AI
- **Architecture**: Node.js/Express backend with vanilla HTML/CSS/JS frontend
- **Key Features**: Multi-platform content generation, engagement scoring, content repurposing

## Development Approach
- Used Kiro for rapid prototyping and iterative development
- Implemented glassmorphism design system for modern UI
- Integrated Google Generative AI for content creation
- Built responsive design with local storage for history management

## Technical Stack
- Backend: Node.js, Express.js, Google Generative AI SDK
- Frontend: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript
- Security: Helmet, CORS, environment variables
- Performance: Compression, optimized static serving

## Key Implementation Details
- Structured AI prompts for consistent JSON responses
- Dynamic engagement scoring with visual feedback
- Local storage for content history management
- Responsive design with slide-out panels
