# AI Content Co-Pilot - Design Specification

## 1. Features (MVP)
- **AI Content Generation**: Transform a simple topic or messy raw notes into polished social media posts.
- **Platform-Specific Optimization**: Tailor content length, hashtags, and formatting for Instagram, LinkedIn, Twitter/X, Threads, Facebook, YouTube, TikTok, and Pinterest.
- **Dynamic Tone & Audience Selection**: Switch between Professional, Casual, Gen-Z, Witty, Academic, and more to hit different demographics (Students, Developers, Entrepreneurs, C-Suite, etc.).
- **One-Click Repurposing**: Instantly convert a LinkedIn post into a Twitter thread or an Instagram caption.
- **AI Engagement Predictor**: Get a 0-100 score based on virality factors, with specific "Why?" reasoning and "How to improve" tips.

## 2. User Flow
1. **Input**: User enters a topic (e.g., "Benefits of AI in Education") or pastes raw text.
2. **Configure**: User selects:
    - Target Platform (e.g., LinkedIn, Threads, TikTok, etc.)
    - Tone (e.g., Professional, Witty & Humorous, Minimalist, etc.)
    - Audience (e.g., Developers, Entrepreneurs, Parents, etc.)
3. **Generate**: User clicks "Generate Magic". AI creates the content.
4. **View**: Output appears in a "Content Card" with the text, hashtags, and CTA.
5. **Score**: Below the content, the "Engagement Score" dial/card shows the predicted success rate and tips.
6. **Repurpose**: User clicks "Repurpose" for a different platform. AI adapts the existing content instantly.

## 4. AI Prompt Engineering

### Main Generation Prompt
```
Act as an expert Social Media Strategist. Your task is to generate high-quality content based on the following parameters:
- Topic/Raw Text: {{input}}
- Platform: {{platform}}
- Tone: {{tone}}
- Audience: {{audience}}

Ensure the output is perfectly formatted for the platform (e.g., hashtags for Instagram, professional polish for LinkedIn).
Return the result in strictly valid JSON format:
{
  "content": "the generated text including emojis and line breaks",
  "hashtags": ["tag1", "tag2"],
  "cta": "the call to action",
  "engagement_score": 85,
  "score_reasoning": "Brief explanation of the score",
  "suggestions": ["suggestion 1", "suggestion 2"]
}
```

### Repurposing Prompt
```
Act as a Content Strategist. I have this content created for {{source_platform}}:
"{{source_content}}"

Repurpose this exact same value and message for {{target_platform}}. 
Adapt the tone, length, and style properly.
Return the result in strictly valid JSON format:
{
  "content": "the repurposed text",
  "hashtags": ["newtag1", "newtag2"],
  "cta": "adapted cta",
  "engagement_score": 80,
  "score_reasoning": "Brief explanation",
  "suggestions": ["suggestion 1"]
}
```

## 5. Engagement Scoring Logic
The score is generated by the AI based on:
- **Hook Strength**: Does the first line grab attention?
- **Platform Fit**: Does the length and style match the platform's best practices?
- **CTA Clarity**: Is there a clear next step for the reader?
- **Trend Relevance**: Does it use current buzzwords or formatting?

## 6. Backend Design
### Endpoints
- `POST /api/generate`
    - Body: `{ input, platform, tone, audience }`
- `POST /api/repurpose`
    - Body: `{ sourceContent, targetPlatform, tone, audience }`

### Logic Flow
1. Receive request.
2. Construct the prompt with variables.
3. Call AI API (OpenAI/Gemini).
4. Parse JSON response.
5. Return JSON to frontend.
